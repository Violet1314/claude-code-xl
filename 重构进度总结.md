## 四、v2.1.0 新增/改动

### 4.1 UI 改进

| 改动项 | 文件 | 说明 |
|--------|------|------|
| **Powerline 状态栏** | `components.py` | 色块分段样式，显示模型/价格/文件数/Token |
| **圆角输入框** | `components.py` | `╭──` / `╰──` 开口圆角边框 |
| **进度动画** | `app.py` | Braille spinner `⠹` + 脉冲条 + 字符计数 |
| **图标更换** | `theme.py` | `⚡` → `◆`（避免与橙色背景混淆）|
| **字体颜色** | `components.py` | 状态栏关键信息改为纯黑 `#000000` |

### 4.2 输入优化

| 改动项 | 文件 | 说明 |
|--------|------|------|
| **命令快捷发送** | `input.py` | `/` 开头命令 Enter 直接发送，普通对话 Enter 换行 |
| **提示符简化** | `input.py` | 移除重复模型名，改为 `│ ❯` |
| **续行对齐** | `input.py` | 多行输入左对齐 `│   ` |

### 4.3 新增命令

| 命令 | 别名 | 说明 |
|------|------|------|
| `/copy` | `/cp` | 导出最后回复中的代码块到文件 |

### 4.4 代码导出功能

**`export_code()` 方法**
- 提取 AI 回复中的所有代码块
- 生成子目录：`data/copy_code/{时间戳}_{标题}/`
- 生成 `README.md`：包含用户需求、代码类型、代码说明
- 生成 `code_N.{ext}`：纯代码文件，带头部注释
- 支持 20+ 语言扩展名自动识别

## 五、模块详细说明

### 5.1 配置模块 (`config/`)

**defaults.py**
- 使用 `@dataclass(frozen=True)` 定义不可变配置
- 包含：`FileDefaults`、`APIDefaults`、`ConversationDefaults`、`UIDefaults`
- 全局单例：`FILE`、`API`、`CONVERSATION`、`UI`
- 版本信息：`VERSION = "2.1.0"`、`APP_NAME`

**settings.py**
- `ModelConfig`：模型配置数据类
- `Settings`：应用配置，包含 `get_model()`、`get_prompt()`、`style_ids` 属性
- `SettingsLoader`：从 `data/config/` 加载 JSON 配置

### 5.2 核心模块 (`core/`)

**client.py**
- `APIClient`：统一 API 客户端
- 支持流式输出（SSE）
- 自动重试机制（指数退避）

**conversation.py**
- `Conversation`：会话管理器
- 核心方法：`get_optimized_messages(max_tokens)` - 倒序优先级修剪

**files.py**
- `FileManager`：文件挂载管理器
- 支持通配符展开（`*.py`、`**/*.js`）

**stats.py**
- `StatsManager`：统计管理器
- 原子写入、增量保存

### 5.3 UI 模块 (`ui/`)

**theme.py**
- `COLORS`：颜色配置（Claude 官方橙色系）
- `ICONS`：图标配置（`◆` 菱形图标）
- `PROMPT_STYLE`：prompt-toolkit 样式

**components.py**
- `show_status_bar()`：Powerline 风格状态栏
- `get_input_border()`：圆角开口边框

**input.py**
- `InputHandler`：输入处理器
- `_create_session()`：命令/对话 Enter 行为分离
- `_get_prompt()`：简化提示符 `│ ❯`
- `_get_continuation()`：续行对齐

### 5.4 命令模块 (`commands/`)

**handlers.py**
- 内置命令：`/help`、`/new`、`/model`、`/style`、`/add`、`/drop`、`/files`、`/refresh`、`/save`、`/history`、`/copy`、`/quit`

### 5.5 主应用 (`app.py`)

**Application 类新增**
- `export_code()`：代码导出主方法
- `_generate_code_title()`：生成代码文件标题
- `_extract_code_description()`：提取代码说明
- `_handle_response()`：脉冲进度条 + 字符计数
