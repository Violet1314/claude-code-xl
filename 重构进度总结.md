## 五、v2.8.3 Edit 工具重构详解

### 匹配策略分层
| 层级 | 策略 | 说明 |
| --- | --- | --- |
| 1 | 精确匹配 | 字符串完全一致 |
| 2 | 忽略空白匹配 | strip() 后比较，容忍缩进差异 |
| 3 | 模糊匹配 | SequenceMatcher 相似度 ≥ 80% |
| 4 | 多候选返回 | 返回所有候选位置 + 相似度分数 |

### 核心方法
```python
# 精确匹配
def _exact_match(content_lines, old_lines) -> List[int]

# 忽略空白匹配
def _whitespace_insensitive_match(content_lines, old_lines) -> List[int]

# 模糊匹配
def _fuzzy_match(content_lines, old_lines, threshold) -> (positions, scores)

# 构建候选报告
def _build_candidates_report(content_lines, positions, n, scores) -> str
```

## 六、工具描述模板（给模型）

### Read
```
读取用户本机文件内容。你可以直接访问用户提供的任何本地路径，无需用户手动粘贴内容。文件会被完整缓存，后续操作使用缓存引用节省 Token。
**重要**：每个文件只需读取一次，系统会自动检测并阻止重复读取。
```

### Edit
```
精确替换文件中的内容。编辑后文件缓存自动更新，无需重新读取。
**多层匹配**：精确匹配 → 忽略空白 → 模糊匹配（相似度 ≥ 80%）。
**失败反馈**：返回候选位置和相似度分数，建议添加更多上下文。
```

### Write
```
创建新文件或覆盖现有文件。慎用，会覆盖已有内容。
```

### Bash
```
执行 shell 命令。当前环境：Windows PowerShell。
注意：必须使用 PowerShell 语法，不支持 Unix 参数。
```
