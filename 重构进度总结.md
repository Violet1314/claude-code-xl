## 五、工具系统（2026-03-25 新增）

### 5.1 工具调用格式

AI 使用 XML 格式调用工具：

```xml
<function_calls>
<invoke name="Read">
<parameter name="file_path">src/app.py</parameter>
</invoke>
</function_calls>
```

### 5.2 内置工具

| 工具 | 参数 | 说明 |
|------|------|------|
| **Read** | `file_path`, `offset?`, `limit?` | 读取文件内容，带行号 |
| **Write** | `file_path`, `content` | 创建或覆盖文件 |
| **Edit** | `file_path`, `old_string`, `new_string` | 精确字符串替换 |
| **Glob** | `pattern`, `path?` | 按文件名模式搜索（支持 `**/*.py`） |
| **Grep** | `pattern`, `path?`, `type?` | 按内容正则搜索 |

### 5.3 权限确认机制

所有工具操作都需要用户授权：
- **Yes (once)**: 仅本次允许
- **Yes (always)**: 本次会话总是允许（缓存权限）
- 按 `Esc` 或 `q` 取消操作

### 5.4 多轮执行

- 单轮最多执行 10 个工具调用
- 最多循环 5 轮（防止死循环）
- 工具结果自动反馈给 AI 继续处理

## 六、核心模块详解

### 6.1 `core/client.py` - API 客户端
- `APIClient`: 统一 API 客户端
- 支持流式输出（SSE）
- 自动重试机制（指数退避，最多 3 次）
- 超时配置：连接 10s，读取 120s

### 6.2 `core/conversation.py` - 会话管理
- `Conversation`: 会话管理器
- `get_optimized_messages()`: 倒序优先级修剪
- 保证保留 system 消息和最后一条用户消息

### 6.3 `core/files.py` - 文件挂载
- `FileManager`: 文件挂载管理器
- 支持 glob 通配符展开
- 限制：单文件 200KB，最多 50 个文件，总字符 800K

### 6.4 `core/stats.py` - Token 统计
- `StatsManager`: 统计管理器
- 会话级统计 + 持久化到 `data/stats/total_stats.json`
- 原子写入保证数据安全

### 6.5 `ui/input.py` - 输入处理
- `InputHandler`: 输入处理器
- `interactive_menu()`: 交互式选择菜单
- 快捷键：Enter 换行（命令直接发送），Esc+Enter 发送

### 6.6 `ui/console.py` - 控制台封装
- 封装 Rich Console
- 提供 `success()`, `error()`, `warning()`, `info()`, `dim()` 等便捷方法
- `markdown()`: 渲染 Markdown 内容
