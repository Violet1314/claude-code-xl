## 四、模块详细说明

### 4.1 配置模块 (`config/`)

**defaults.py**
- 使用 `@dataclass(frozen=True)` 定义不可变配置
- 包含：`FileDefaults`、`APIDefaults`、`ConversationDefaults`、`UIDefaults`
- 全局单例：`FILE`、`API`、`CONVERSATION`、`UI`
- 版本信息：`VERSION = "2.0.0"`、`APP_NAME`

**settings.py**
- `ModelConfig`：模型配置数据类
- `Settings`：应用配置，包含 `get_model()`、`get_prompt()`、`style_ids` 属性
- `SettingsLoader`：从 `data/config/` 加载 JSON 配置
- 支持兜底配置（配置文件不存在时）

### 4.2 核心模块 (`core/`)

**client.py**
- `APIClient`：统一 API 客户端
- 支持流式输出（SSE）
- 自动重试机制（指数退避）
- 超时配置从 `defaults.py` 读取
- 静态方法：`extract_content()`、`extract_usage()`
- 支持上下文管理器（`with` 语句）

**conversation.py**
- `Message`：消息数据类
- `Conversation`：会话管理器
- 核心方法：`get_optimized_messages(max_tokens)` - 倒序优先级修剪
- 保证：始终保留 system 消息 + 最后一条用户消息
- 支持：`reset()`、`load_messages()`

**files.py**
- `AttachedFile`：挂载文件数据类
- `FileManager`：文件挂载管理器
- 支持通配符展开（`*.py`、`**/*.js`）
- 验证：大小限制、扩展名、隐藏文件
- 方法：`add()`、`drop()`、`refresh()`、`build_context()`

**stats.py**
- `SessionStats`：会话统计数据类
- `StatsManager`：统计管理器
- 原子写入（防止数据损坏）
- 增量保存（避免重复计算）

### 4.3 UI 模块 (`ui/`)

**theme.py**
- `COLORS`：颜色配置字典（Claude 官方橙色系）
- `ICONS`：图标配置
- `PROMPT_STYLE`：prompt-toolkit 样式
- `LOGO_GRADIENT`：Logo 渐变色数组

**console.py**
- 全局 `_console` 单例
- 状态消息：`success()`、`error()`、`warning()`、`info()`、`dim()`
- 内容渲染：`markdown()`、`code()`
- 布局：`rule()`、`blank()`、`clear()`

**components.py**
- `show_logo()`：渐变色 ASCII Logo
- `show_welcome()`：欢迎界面
- `show_status_bar()`：底部状态栏
- `show_model_list()`、`show_style_list()`、`show_files_list()`、`show_history_list()`
- `get_input_border()`：输入框边框

**input.py**
- `CommandCompleter`：命令自动补全
- `InputHandler`：输入处理器（延迟初始化 PromptSession）
- `interactive_menu()`：交互式选择菜单（上下键 + 数字快捷键）
- `input_number()`：数字输入

**renderer.py**
- `render_response()`：渲染 AI 响应（带分割线和耗时统计）

### 4.4 命令模块 (`commands/`)

**base.py**
- `Command`：抽象基类
- 属性：`name`、`description`、`aliases`
- 抽象方法：`execute(args)`

**registry.py**
- `CommandRegistry`：命令注册表
- 支持别名、大小写不敏感
- 方法：`register()`、`get()`、`execute()`、`has()`、`list_commands()`

**handlers.py**
- 内置命令：`/help`、`/new`、`/model`、`/style`、`/add`、`/drop`、`/files`、`/refresh`、`/save`、`/history`、`/quit`
- `BUILTIN_COMMANDS` 列表

### 4.5 工具模块 (`utils/`)

**tokens.py**
- `estimate_tokens(text)`：单文本估算
- `estimate_messages_tokens(messages)`：消息列表估算
- 系数：中文 1.5，英文 0.25

**paths.py**
- `resolve_path()`：路径解析（去引号、转绝对路径）
- `is_hidden()`、`is_supported_extension()`、`get_extension()`
- `expand_glob()`：通配符展开
- `SUPPORTED_EXTENSIONS`：支持的文件扩展名集合

### 4.6 主应用 (`app.py`)

**Application 类**
- 整合所有模块
- 初始化：加载配置、创建客户端、注册命令
- 对话：`chat()`、`_handle_response()`
- 文件：`add_files()`、`drop_files()`、`show_files()`、`refresh_files()`
- 会话：`save_conversation()`、`load_history()`、`reset_conversation()`
- 切换：`select_model()`、`select_style()`
- 主循环：`run()`
